## Dock Customs
- This bot has been designed for python version 14.0.3, discord python version 2.7.0, and ro.py/roblox version 2.0.0. 

### Packages
- Make sure to install python to start. Nothing will work without python!
- Then in a terminal whether on a code platform such as GitHub or Visual Studio Code, or on your computer's terminal run the following:
- `pip install discord.py`
- `pip install dotenv`
- `pip install ro.py`
- `pip install roblox`
If one of these steps is skipped or not done correctly it will fail 100% of the time. 

This is a discord design bot meant for Roblox design on the specific game of Emergancy Response: Liberty County. Most if not all of the categories in the deisgn panel relate towards ER: LC.
